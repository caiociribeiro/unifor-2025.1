Lista de Exercicios - Javascript

1. O que é o JavaScript?

O JavaScript é uma linguagem de programação usada principalmente como uma linguagem de script para páginas web. Interage com o HTML, sendo responsavel pela parte interativa de sites.

2. Qual a diferença entre var, let e const?

var: Possui escopo de funcao. Pode ser redeclarada.
let: Possui escopo de bloco ({}). Nao pode ser redeclarada no mesmo bloco, mas pode ser reatribuida.
const: Possui escopo de bloco. Nao pode ser redeclarada e nem reatribuida. (objetos declarados com const podem ter as suas propriedades reatribuidas.)

3. O que são funções em JavaScript?

Funcoes sao "subprogramas" que podem ser chamados por codigo externo (ou interno no caso de recursao).
Funcoes tradicionais: Declaradas com a keyword function. Possuem seu próprio contexto this.
Arrow functions: Utilizam a sintaxe () => {} e herdam o contexto this do escopo onde foram definidas.

4. O que é o DOM (Document Object Model)?

DOM é a representação de dados dos objetos que compõem a estrutura e o conteúdo de um documento na Web. Permite que linguagens como o JavaScript acessem, modifiquem e atualizem seus elementos.

5. Como funciona a execução síncrona e assíncrona no JavaScript?

Código Síncrono: Executa uma instrução de cada vez, na ordem em que aparecem.
Código Assíncrono: Permite a execução de operações sem bloquear o fluxo principal, permitindo requisições a servidores sem travar a interface do usuário, por exemplo.

6. O que são Promises e como elas são usadas?

Uma Promise é um objeto que representa o eventual sucesso (ou falha) de uma operação assíncrona. Ela permite encadear operações e tratar erros de forma mais organizada.

7. Explique o uso de async e await em JavaScript.

async e await são sintaxes que simplificam o trabalho com promises, permitindo escrever código assíncrono de forma mais parecida com código síncrono.

8. Qual a diferença entre for...in e for...of?

for in: Itera sobre as propriedades enumeráveis de um objeto, ou indices de uma array.
for off: Itera sobre valores iteráveis, como arrays, strings, map, set e etc.

9. O que é desestruturação (destructuring) em JavaScript?

Desestruturação permite extrair valores de arrays ou propriedades de objetos em variáveis distintas.

Em arrays:

                    
const numeros = [1, 2, 3];
const [a, b, c] = numeros;
console.log(a, b, c); // 1 2 3

Em objetos:

                
const pessoa = { nome: "Ana", idade: 25 };
const { nome, idade } = pessoa;
console.log(nome, idade); // Ana 25

10. O que são os operadores spread e rest em JavaScript?

Spread: Expande elementos de um array ou objeto.
Rest: Permite representar um número indefinido de argumentos na forma de um Array

Ex. 1:

                    
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5];
console.log(arr2); // // [1, 2, 3, 4, 5]

Ex. 2:

                    
function soma(...numeros) {
  return numeros.reduce((ac, num) => ac + num, 0);
}
console.log(soma(1, 2, 3, 4)); // 10
